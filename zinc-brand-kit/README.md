## Wordmark Anatomy

```
┌─ SCRAM ──────────── AZ-30 ─┐
│  Zinc          a-Z30        │
└──────── panic ──────────────┘
```

- **SCRAM** — nuclear emergency shutdown term
- **AZ-30** — references AZ-5 (Chernobyl SCRAM button); 30 = atomic number of Zinc (Zn)
- **Zinc** — Black Ops One, the display face
- **a-Z30** — periodic/part-number callout; small `a`, large `Z`, 30
- **panic** — Go's runtime fatal error keyword; since Zinc transpiles to Go

## Mascot — Hazmat Badger

The badger hunts gophers. Zinc transpiles to Go (the gopher language).

**Face markings (real badger anatomy, brand colours):**
- White central crown — runs nose to crown
- Two zinc-blue stripes — sweep from snout through/around each eye up over the head
- Amber eyes sit inside the blue stripe bands

**Gear:**
- Soviet-yellow hazmat suit with horizontal seam bands
- Hard hat with flipped-up tinted visor
- Breathing mask straps + filter canister
- Blinking green dosimeter (left chest)
- Radiation trefoil (right chest, warning yellow)
- Dark rubber gloves + heavy boots

## Usage Notes

- Minimum size: 44px tall for the lockup
- On dark backgrounds: use the inverted stamp (parchment border/type on dark)
- The ink displacement filter (`feDisplacementMap`) on the wordmark is decorative — safe to remove for flat/print use
- SVGs are resolution-independent; scale to any size
