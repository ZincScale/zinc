# Zinc

Thin transpilers that remove syntax warts from target languages. Clean source in, clean output you can read and edit directly.

| Project | Target | Status |
|---------|--------|--------|
| [zinc-go](zinc-go/) | Go | Active — 25 examples, typed channels, classes, streams with loop fusion |
| [zinc-python](zinc-python/) | Python 3.14t | Complete — self-hosting, braces syntax, auto f-strings |
